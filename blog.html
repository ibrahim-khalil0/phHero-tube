Scope of var, let, and const

1. var is a global scope. After declaring it can be accessed from anywhere. It can be redeclared and reassigned.

2. let is a block scope. After declaring it in a function it can't be access outside of function. It can't be redeclared but can reassigned.

3. const is a block scope. After declaring it in a function it can't be access outside of function. It can't be redeclared and reassigned.

Use cases of null and undefined

If the value of something or the value of a variable is not defined, it will show undefined. (example: if i declared a variable but not initialized, if i declare a function but don't return anything)

Null takes the place of the value of a variable or object
